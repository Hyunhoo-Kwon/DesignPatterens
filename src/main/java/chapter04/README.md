# chapter04. 구조 패턴 (Structural Patterns)

### 정의
- 더 큰 구조를 형성하기 이해 어떻게 클래스와 객체를 합성하는가와 관련된 패턴
  - 구조 클래스 패턴: 상속을 이용하여 인퍼테이스나 구현을 복합
  - 구조 객체 패턴: 구현을 복합하는 것이 아닌 새로운 기능을 실현하기 위해 객체를 합성하는 방법
